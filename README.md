# Url Shortener

將原始網址轉為短網址。

## 環境

- windows 10
- body-parser: ^1.19.0,
- connect-flash: ^0.1.1,
- express: ^4.17.1,
- express-handlebars: ^3.1.0,
- mongoose: ^5.6.7,
- nodemon: ^1.19.1

## 下載安裝

使用終端機操作

1. 下載資料夾
2. 進入資料夾
3. 安裝套件

```
git clone https://github.com/cTaohe/AC3A20Q4.git
cd AC3A20Q4/
npm install
```
## 本地運行

終端機操作
```
npm run dev
```
瀏覽器輸入

[http://localhost:3000/](http://localhost:3000/)
